\section{Open Access  Computer Programs developed by Petros Souvatzis}\\
\cvline{I.}{The code scph, exclusively developed by Petros Souvatzis,  in which the Self Consistent Ab Initio Lattice Dynamical method  (SCAILD) has been implemented. The code and corresponding manual can be found at:} \\
\cvline{}{\texttt{http://www.uquantchem.com/scaild.html}}\\ \\
\cvline{II.}{The uquantchem code (Uppsala Quantum Chemistry package) is a program exclusively developed by P. Souvatzis. The code is  designed to solve the non-relativistic Schr{\"o}dinger equation for atoms and molecules using gaussian basis sets.
Amongst the capabilities are:} \\
\cvline{}{(1)  Unrestricted Hartree Fock (URHF)}
\cvline{}{(2)  Restricted Hartree Fock (RHF)}
\cvline{}{(3)  Configuration Interaction calculations including Singles and Doubles (CISD)}
\cvline{}{(4)  M{\o}ller-Plesset manybody perturbation calculations to 2:nd order (MP2)}
\cvline{}{(5)  Diffusion Quantum Monte Carlo (DQMC)}
\cvline{}{(6)  Variational Quantum Monte Carlo (VMC)}
\cvline{}{(7)  Density Functional Theory (DFT) calculations}
\cvline{}{(8)  Time Dependent Density Functional Theory (TDDFT) calculations}
\cvline{}{(9)  Extended Lagrangian Born-Oppenheimer Molecular Dynamics (XL-BOMD)}
\cvline{}{(10)  Fast First Principles Molecular Dynamics (FFP-MD)}
\cvline{}{(11) OPENMP and MPI parallelization}\\
\cvline{}{The code and corresponding manual can be found at:} \\
\cvline{}{\texttt{http://www.uquantchem.com/uquantchem.html}}
